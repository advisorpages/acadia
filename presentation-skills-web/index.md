## How to Use This Training

**For Practice:** Read through each chunk to understand the flow and key talking points.

**During Meetings:** Use this as a quick reference for specific sections or delivery notes.

**For Team Training:** Share this link with new team members to help them learn the presentation structure.

## Key Concepts

- **Financial House Metaphor:** Using the structure of a house to explain financial planning
- **Non-Sales Approach:** Positioning yourself as a resource, not a salesperson
- **Value-First:** Demonstrating value before asking for anything
- **Three Outcomes:** Understanding that each conversation has multiple positive outcomes
