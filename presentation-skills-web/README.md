# Presentation Skills Training Website

This is a simple GitHub Pages site containing the Financial House and Power of PAC presentation training materials.

## Purpose

This website provides team members with easy access to:
- Individual script chunks for quick reference
- Complete presentation script
- Navigation between different sections
- Mobile-friendly design for on-the-go access

## Structure

- **`index.md`** - Homepage with overview and navigation
- **`chunks/`** - Individual script chunks organized by section
- **`complete-script.md`** - Full presentation in one document
- **`assets/css/`** - Simple styling for readability
- **`_config.yml`** - Jekyll configuration

## How to Use

1. **Local Development:** Use `bundle exec jekyll serve` to run locally
2. **GitHub Pages:** Enable GitHub Pages in repository settings
3. **Updates:** Edit markdown files and push to update the site

## Features

- ✅ Mobile-responsive design
- ✅ Fast loading (no heavy frameworks)
- ✅ Easy navigation between sections
- ✅ Clean, readable layout
- ✅ Simple maintenance (just markdown)

## Customization

To customize the site:
- Edit `_config.yml` for site settings
- Modify `assets/css/style.css` for styling
- Update content in markdown files
- Add new chunks by creating new files in `chunks/` directory

## Deployment

This site is designed for GitHub Pages and will work automatically when:
1. Pushed to a GitHub repository
2. GitHub Pages is enabled in repository settings
3. Source is set to "Deploy from a branch" (usually `main`)
